## 🛠️ Installation & Setup

### 🔧 System Requirements
- **PHP**: 8.2 or higher
- **Node.js**: 18.0 or higher
- **MySQL**: 8.0 or higher
- **Redis**: 6.0 or higher
- **Composer**: Latest version
- **NPM/Yarn**: Latest version

### 🚀 Quick Installation
```bash
# Backend Setup
cd agency-backend
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate --seed
php artisan serve

# Frontend Setup
cd agency-frontend
npm install
npm run build
npm start
```
